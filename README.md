# GameArena
GameArena is a unified DAA visualization platform integrating Maze Escape, Spy Mission, and Puzzle Solver modules to demonstrate real-time execution of algorithms like BFS, DFS, Dijkstra, A, and Backtracking using interactive visual analytics.

## 🎯 Games
- **Maze Escape** — BFS, DFS, Dijkstra, A*
- **Puzzle Solver** — A*, Branch & Bound
- **Spy Mission** — Dijkstra, Backtracking

## 🛠️ Tech Stack
HTML, CSS, JavaScript, Canvas API
